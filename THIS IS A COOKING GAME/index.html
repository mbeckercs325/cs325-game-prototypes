THIS IS A COOKING GAME

  • High Concept: THIS IS A COOKING GAME is a cooperative cooking game.
  • Theme: None, I wanted to make my own game.
  • Player Experience Goals: Players will have to cooperate in order to complete as many recipes as they can.
  • Gameplay: The goal of the game is to work together to create as many burgers as possible. A burger consists of cooked meat, chopped lettuce and tomato, a bun, and a plate. Players have to assemble all of these before they can deliver the order.
    Player 1 Controls - WASD, E to select
    Player 2 Controls - Arrow Keys, ? to select
